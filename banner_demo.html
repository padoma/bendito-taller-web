Guía de Modificaciones para este Archivo

  • Modificar el Texto: Ve a las líneas 186 y 187 de este código HTML. Edita las frases dentro de las etiquetas <p class="cursive-text">...</p> para cambiar el mensaje de tu banner.
  • Cambiar el Color del Banner: En la línea 47 del CSS, localiza la regla background-color: #d19e93;. Puedes cambiar #d19e93 por cualquier otro color hexadecimal que desees.
  • Ajustar la Velocidad del Balanceo (Viento): En las líneas 94 y 101, modifica los valores de tiempo (ej. cambiar 10s o 8.5s por valores menores como 5s para movimiento rápido, o mayores como 15s para movimiento ultra lento).
  • Ajustar el Color de las Hojas: En las etiquetas <svg> de las hojas (líneas 152 y 191), puedes cambiar el parámetro stroke="rgba(255,255,255,0.85)" por cualquier color (por ejemplo, stroke="rgba(28,28,28,0.7)" si quieres hojas oscuras en lugar de blancas).
